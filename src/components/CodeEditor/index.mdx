# CodeEditor

## Basic usage

<Playground>
  <CodeEditor />
</Playground>

## Properties

| 属性 | 说明 | 类型 | 默认值 |
| ---- | ---- | ---- | ---- |
| className | 自定义样式 | string，只支持 CSS Module | 无 |
| mode | CodeEditor 语言格式 | string | 'yaml' |
| options | react-ace 支持的属性 | object | {} |
| onChange | 改变 CodeEditor 值的回调 | func | function() {} |
| value | CodeEditor 的值 | string 或 object | '' |
