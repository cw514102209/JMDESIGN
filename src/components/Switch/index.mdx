## Properties

| 属性 | 说明 | 类型 | 默认值 |
| ---- | ---- | ---- | ---- |
| className | 自定义样式 | string，只支持 CSS Module | 无 |
| disabled |  | bool | 无 |
| type | 开关的类型，可选值为 'default'，'primary'，'control' | string | 'default' |
| text | 开关打开时显示的文字 | string | 无 |
| checked | 开关是否打开 | bool | false |
| onChange | 切换开关的回调 | func | function() {} |
